如果有 msgpack-rpc-python 和 msgpack-python，先使用pip uninstall 卸载。 
pip uninstall msgpack-rpc-python
pip uninstall msgpack-python

1. 安装最新msgpack：
pip install msgpack

2. 安装msgpack-rpc-python 修正版：
cd install_guide
pip install msgpack_rpc_python-0.4-py3-none-any.whl

3.安装airsim 修正版：
pip install airsim-1.8.1-py3-none-any.whl
